1. 一个基于goja的动态代码沙盒
2. 集成了chromedp，可以进行浏览器操作
3. 集成http请求，可以进行http请求
4. 支持本地文件系统的文件操作，可以进行文件读写（使用GO扩展）
    1. 使用操作系统默认软件打开文件，比如txt、pdf、word、excel、图片、视频等
    2. 读取文件元信息（创建时间、修改时间、文件类型、大小、文件重命名）
    3. 读取文件内容（文档、视频、图片等，支持分页读取
    4. 读取文本文件的前几行、后几行
    5. 读取文件的md5/sha1/sha256/sha512等哈希值
    6. 读取word文件内容，支持分页读取
    7. 读取excel文件内容，支持分页读取
    8. 读取ppt文件内容，支持分页读取
    9. 读取pdf文件内容，支持分页读取
    10. 读取图片文件的base64编码
5. 可以作为大模型的外挂执行器，可以执行大模型返回的代码完成一些列操作
6. 支持一些系统操作，比如：
    1. 获取当前时间
    2. 获取当前日期
    3. 获取cpu数量
    4. 获取内存大小
    5. 获取磁盘空间大小
7. 作为go库可以引入到其他项目中去
